# ProyectoHeDE

Este proyecto fue creado con el propósito de brindar al colegio Abelardo Gamarra una página web estática, ya que actualmente no cuenta con una.

Desarrollaremos esta página utilizando **HTML, CSS y JavaScript**.

Además, este trabajo nos permite aprender cómo se trabaja de manera colaborativa y remota usando herramientas como Git y GitHub desde la línea de comandos.

##Estructura del Proyecto

- `/images`: Recursos multimedia
- `/javascript`: Archivos del tipo javascript
- `/styles`: Archivos css
- En la carpeta raíz del proyecto se encuentran  los archivos html
